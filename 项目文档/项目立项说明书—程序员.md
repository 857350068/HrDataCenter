# <a name="_toc396126158"></a>**1 Project Proposal 项目提出**
1. <a name="_toc396126159"></a>**Project Brief 项目简介**

本项目是人力资源数据中心系统（HrDataCenter），基于Spring Boot + Vue 3的前后端分离架构，用于企业人力资源数据管理和分析。系统整合了员工档案管理、数据分析、预警规则、报表模板等功能，支持MySQL和Hive双数据源，为企业提供全面的人力资源数据支撑。

1. <a name="_toc396126160"></a>**Project Goal 项目目标**

项目目标包括：
1. 建立统一的人力资源数据管理平台，实现员工档案的数字化管理
2. 提供多维度数据分析功能，包括组织效能、人才梯队、薪酬福利等八大分析模块
3. 实现数据预警机制，通过自定义规则及时发现人力资源风险
4. 支持报表模板管理，满足不同层级管理者的数据查询需求
5. 建立完善的权限管理体系，确保数据安全和访问控制

1. <a name="_toc396126161"></a>**System Scope 系统边界**

系统包含以下核心功能模块：
- 用户认证与授权：登录、注册、权限管理
- 员工档案管理：员工信息的增删改查、批量导入导出
- 数据分类管理：数据目录、元数据管理
- 数据同步管理：MySQL到Hive的数据同步
- 数据分析功能：薪酬福利、员工流失、绩效管理等八个分析模块
- 报表模板管理：自定义报表模板
- 预警规则管理：数据异常预警
- 收藏夹功能：常用数据收藏
- 管理员功能：用户管理、数据同步控制

   1. <a name="_toc396126167"></a>**Technical Risks 技术风险**

(1)Hive大数据技术没有深入学习过

(2)前后端分离架构的JWT认证实现可能存在安全隐患

Resolution 解决:

(1)通过官方文档和在线课程学习Hive技术，参考Apache Hive官方文档和实战案例

(2)参考Spring Security官方文档和JWT最佳实践，确保token的安全性和有效期管理
